# ZILA Website - Development Plan

## Design Guidelines

### Design References
- **Futuristic Web3/Crypto Sites**: Dark mode with neon accents
- **Glassmorphism UI**: Semi-transparent cards with blur effects
- **3D Interactive Elements**: Floating, rotating token animations

### Color Palette
- Primary Background: #0a0a0f (Deep Black)
- Secondary Background: #12121a (Dark Purple-Black)
- Card Background: rgba(20, 20, 35, 0.7) (Glassmorphism)
- Primary Accent: #8b5cf6 (Purple)
- Secondary Accent: #3b82f6 (Blue)
- Neon Glow: #a855f7 (Bright Purple)
- Neon Blue: #06b6d4 (Cyan)
- Text Primary: #ffffff (White)
- Text Secondary: #a1a1aa (Gray)

### Typography
- Heading Font: 'Orbitron', sans-serif (Futuristic)
- Body Font: 'Inter', sans-serif (Clean, readable)
- Heading1: 48px, weight 700
- Heading2: 36px, weight 600
- Heading3: 24px, weight 600
- Body: 16px, weight 400

### Key Component Styles
- **Buttons**: Gradient purple-blue, glow on hover, rounded corners
- **Cards**: Glassmorphism with blur, neon border on hover
- **Navigation**: Fixed, glassmorphism background
- **Animations**: Smooth 0.3s transitions, floating keyframes

## Development Tasks

1. **Setup & Structure** - Create HTML structure with all sections and navigation
2. **Generate Images** - Create all 9 images using ImageCreator
3. **CSS Styling** - Implement dark theme, glassmorphism, neon effects, animations
4. **JavaScript Interactivity** - 3D token animation, pie chart, scroll animations, navigation
5. **Home Page** - Hero section, token description, ecosystem preview, team section
6. **Ecosystem Page** - Detailed ecosystem explanation with visual diagrams
7. **Airdrop Page** - 3D token, registration CTA
8. **Tokenomics Page** - Interactive pie chart with clickable segments
9. **Roadmap Page** - 3D timeline with floating cards
10. **Footer** - Links, security, disclaimer, copyright
